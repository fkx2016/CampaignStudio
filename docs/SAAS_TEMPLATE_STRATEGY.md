## ðŸ—ºï¸ Deployment Roadmap for Campaign Studio

### Phase 1: The "MVP" Deploy (Current Goal)
1.  **Database:** Create Supabase Project -> Get `DATABASE_URL`.
2.  **Backend:**
    *   Push to GitHub.
    *   Connect Railway to Repo.
    *   Set `ROOT_DIRECTORY` to `/backend`.
    *   Add `DATABASE_URL` env var.
3.  **Frontend:**
    *   Connect Vercel to Repo.
    *   Set `ROOT_DIRECTORY` to `/next-gen-app`.
    *   Add `NEXT_PUBLIC_API_URL` (Railway URL).

### Phase 2: The "Scale" Deploy
*   Add **Redis** (on Railway) for task queues (Celery).
*   Add **CDN** (Cloudflare) for media caching.
