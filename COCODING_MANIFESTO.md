## üéì Getting Started

### Prerequisites:
1. Docker installed
2. Git proficiency
3. TypeScript/Pydantic knowledge
4. AI assistant (Claude, ChatGPT, etc.)

### First Session:
1. Define a small, clear task
2. Create a safety branch
3. Work iteratively with AI
4. Test after each change
5. Commit when working
6. Document what you learned

### Key Mindset Shifts:
- AI is a **partner**, not a tool or replacement
- Mistakes are **learning opportunities**, not failures
- Documentation is **investment**, not overhead
- Testing is **confidence**, not bureaucracy
